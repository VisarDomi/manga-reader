Provider Boundary Mismatch

Observed in source: provider source is small (806 LOC), while provider-specific runtime policy lives mostly in server files: CacheService, BrowserSession, routes/search.ts, ByteCacheService.

Inference: the provider boundary is still not as strong as the product model wants. Adding Mangadot made the boundary clearer, but much provider behavior is still expressed as conditionals around generic services rather than provider-owned capabilities.
